# Library Management System

## Project Description

This is a Python-based Library Management System that helps manage books efficiently using dictionary-based book records.

The system allows users to:
- Add new books
- Display available books
- Issue books with student details
- Return books
- Calculate late return fines automatically
- Improve interaction with user-friendly menu options

## Features

### 1. Dictionary-Based Book Records
Books are stored using Python dictionaries for better organization and faster access.

### 2. Book Issue System
Stores:
- Student Name
- Issue Date
- Return Duration

### 3. Late Return Fine Calculation
Progressive fine charges are applied:
- First week late → small fine
- Additional weeks → increasing fine

### 4. Better User Interface
Clear menu-driven interaction for smooth user experience.
